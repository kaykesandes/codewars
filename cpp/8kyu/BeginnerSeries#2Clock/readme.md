<h1>Beginner Series #2 Clock</h1>

<p>
    Clock shows <code>h</code> hours, <code>m</code> minutes and <code>s</code> seconds after midnight. <br> Your task is to write a function which returns the time since midnight in milliseconds.
</p>

<h2>Example:</h2>
<code>
    h = 0<br>
    m = 1<br>
    s = 1<br>
<br>
    result = 61000<br>
</code>
